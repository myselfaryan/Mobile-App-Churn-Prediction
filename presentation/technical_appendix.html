Technical Appendix

Data Generation

Synthetic dataset of 50,000 user records created with realistic correlations between features and churn using Python and NumPy.

Machine Learning Pipeline

Feature engineering, imputation, and scaling using Scikit-learn. Models evaluated: Logistic Regression, Random Forest, and XGBoost.

Segmentation Methodology

User segmentation performed using K-means clustering with k=5 clusters based on behavioral and demographic features.

A/B Testing Simulation

Statistical significance tested using chi2_contingency from SciPy. Business metrics calculated using Pandas for data manipulation.
